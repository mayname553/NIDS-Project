2. ✅ 确认模型已训练：`backend/models/*.pkl` 和 `*.keras`
3. ✅ 确认依赖已安装：`pip install pandas numpy matplotlib seaborn scikit-learn tensorflow`
4. ✅ 确认有足够磁盘空间（至少 2GB）

### 常见问题

1. **找不到数据集**：脚本会自动搜索多个可能位置
2. **找不到模型**：先运行训练脚本生成模型
3. **内存不足**：减少 RandomForest 的 n_estimators 参数
4. **图表中文乱码**：安装中文字体（SimHei）
