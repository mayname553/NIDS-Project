## 📁 文件清单

### 1. 样本平衡实验
- **文件**: `backend/check_balance.py`
- **功能**: 对比原始数据、随机欠采样、随机过采样三种策略
- **输出**: 
  - `outputs/balance_experiment_results.csv` - 实验结果表
  - `outputs/experiment_comparison.png` - 指标对比图
  - `outputs/class_distribution_comparison.png` - 类别分布图
  - `outputs/sample_count_comparison.png` - 样本数量对比图

### 2. 模型对比实验
- **文件**: `evidence/05_model_comparison.py`
- **功能**: 对比 RandomForest、MLP(Hybrid)、CNN-LSTM 三个模型
- **输出**:
  - `evidence/output/model_comparison_results.csv` - 性能对比表
  - `evidence/output/model_comparison_detailed.csv` - 详细结果（含混淆矩阵）
  - `evidence/output/charts/model_performance_comparison.png` - 性能对比图
  - `evidence/output/charts/confusion_matrices.png` - 混淆矩阵对比
  - `evidence/output/charts/performance_radar.png` - 性能雷达图
  - `evidence/output/charts/feature_importance.png` - 特征重要性图

### 3. 图表生成工具
- **文件**: `evidence/generate_charts.py`
- **功能**: 统一生成所有可视化图表
- **特性**: 
  - 混淆矩阵生成
  - 模型性能对比图
  - 性能雷达图
  - 特征重要性图
  - 训练历史图（深度学习模型）

确保已安装所有依赖：

```bash
pip install pandas numpy scikit-learn matplotlib seaborn tensorflow
```

确保数据集位于以下路径之一：
- `backend/dataset/KDDTrain+.txt`
- `backend/dataset/KDDTest+.txt`

## 📝 实验结果解读

### 样本平衡实验

**目的**: 验证样本平衡策略对模型性能的影响

**结论**:
- NSL-KDD 数据集在二分类任务上已经相对平衡
- 样本平衡策略对性能提升有限（通常 < 2%）
- 建议直接使用原始数据集进行训练

**注意事项**:
- 过采样可能导致过拟合（重复样本）
- 欠采样会丢失部分信息（样本减少）
- 可考虑更高级的方法：SMOTE、ADASYN 等

### 模型对比实验

**目的**: 对比不同模型架构的性能

**预期结果**:
1. **RandomForest**: 准确率最高，训练速度快，可解释性强
2. **MLP(Hybrid)**: 性能中等，训练时间适中
3. **CNN-LSTM**: 捕捉时序特征，但训练时间较长

**选型建议**:
- **生产环境**: 推荐 RandomForest（速度快、稳定性好）
- **研究实验**: 可尝试 CNN-LSTM（探索深度学习潜力）
- **资源受限**: 使用 MLP(Hybrid)（平衡性能与资源）

## 🔧 故障排除

### 问题1: 找不到数据集

**错误信息**: `找不到数据集文件！`

**解决方法**:
```bash
# 确保数据集在正确位置
ls backend/dataset/KDDTrain+.txt
ls backend/dataset/KDDTest+.txt

# 如果不存在，从 NSL-KDD 官网下载
# https://www.unb.ca/cic/datasets/nsl.html
```

### 问题2: 找不到模型文件

**错误信息**: `找不到模型文件目录！`

**解决方法**:
```bash
# 先训练模型
cd backend
python train_model.py          # 训练 RandomForest
python train_hybrid.py         # 训练 MLP
python train_cnn_lstm.py       # 训练 CNN-LSTM

# 确认模型已保存
ls backend/models/
```

### 问题3: 缺少依赖库

**错误信息**: `ModuleNotFoundError: No module named 'xxx'`

**解决方法**:
```bash
pip install pandas numpy scikit-learn matplotlib seaborn tensorflow
```

### 问题4: 内存不足

**错误信息**: `MemoryError` 或程序卡死

**解决方法**:
- 减少 RandomForest 的 `n_estimators` 参数（默认 50）
- 减少深度学习模型的 `batch_size`
- 使用更小的数据集子集进行测试
