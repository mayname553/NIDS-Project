如果上传失败，查看错误提示信息，检查文件格式和网络连接。

## 🔧 故障排除

### 问题1: 上传按钮不可用
**原因**: 未选择文件  
**解决**: 先选择一个有效的 PCAP 文件

### 问题2: 上传失败
**原因**: 后端服务未启动或 API 错误  
**解决**: 
1. 检查后端服务是否运行
2. 查看浏览器控制台错误信息
3. 查看后端日志

### 问题3: 历史记录不显示
**原因**: localStorage 被禁用  
**解决**: 检查浏览器设置，启用 localStorage

### 问题4: 进度条不动
**原因**: 网络连接问题  
**解决**: 检查网络连接，刷新页面重试

## 💻 技术细节

### API 接口
```
POST /api/upload
Content-Type: multipart/form-data
Body: file (PCAP文件)

成功响应:
{
  "success": true,
  "filename": "uploaded_file.pcap",
  "message": "文件上传成功"
}

失败响应:
{
  "success": false,
  "message": "错误信息"
}
```

### 本地存储
```javascript
// 历史记录存储在 localStorage
localStorage.uploadHistory = [
  {
    filename: "traffic.pcap",
    size: "2.5 MB",
    status: "success",
    message: "上传成功",
    timestamp: "2025-05-11T14:30:00.000Z"
  },
  // ... 最多20条记录
]
```
