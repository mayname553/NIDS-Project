**预计完成时间**：今晚（约30-60分钟）

**训练配置**：
- 数据集：NSL-KDD
- 训练集：125,973 样本
- 测试集：22,544 样本
- 特征数：41
- Epochs：50
- Batch size：128
- 优化器：Adam
- 学习率：0.001

**模型架构**：
```
CNN层：
- Conv1D(64 filters, kernel_size=3) + ReLU + MaxPooling
- Conv1D(128 filters, kernel_size=3) + ReLU + MaxPooling

LSTM层：
- LSTM(64 units, return_sequences=True)
- LSTM(32 units)

全连接层：
- Dense(64) + ReLU + Dropout(0.5)
- Dense(32) + ReLU + Dropout(0.3)
- Dense(1) + Sigmoid
```

**回调函数**：
- EarlyStopping（监控 val_loss，patience=10）
- ReduceLROnPlateau（监控 val_loss，patience=5）
- ModelCheckpoint（保存最佳模型，监控 val_recall）
- TensorBoard（日志记录）
