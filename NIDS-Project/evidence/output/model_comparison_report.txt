二、模型架构说明
------------------------------------------------------------
  1. RandomForest（随机森林）
     - 算法类型：集成学习（Bagging）
     - 决策树数量：100棵
     - 最大深度：20层
     - 特点：可解释性强，训练速度快

  2. MLP（多层感知机）
     - 算法类型：深度学习（全连接神经网络）
     - 隐藏层结构：64 -> 32
     - 激活函数：ReLU
     - 特点：非线性拟合能力强

  3. CNN-LSTM Hybrid（混合模型）
     - 算法类型：深度学习（卷积 + 循环神经网络）
     - CNN层：2层（64 -> 128 filters）
     - LSTM层：2层（64 -> 32 units）
     - 特点：能提取空间和时序特征

五、各模型优缺点分析
------------------------------------------------------------
  1. RandomForest（随机森林）
     优点：
     - 训练和推理速度快
     - 模型可解释性好（特征重要性）
     - 对参数调优要求相对较低
     - 不容易过拟合
     缺点：
     - 难以捕捉复杂的非线性关系
     - 对高维稀疏数据效果有限

  2. MLP（多层感知机）
     优点：
     - 能学习复杂的非线性关系
     - 训练相对快速（相比CNN-LSTM）
     - 对特征学习有一定自动性
     缺点：
     - 对特征工程有一定依赖
     - 容易过拟合，需正则化

  3. CNN-LSTM Hybrid（混合模型）
     优点：
     - CNN能提取局部空间特征
     - LSTM能捕捉时序依赖关系
     - 混合架构适合复杂模式识别
     - 在本数据集上表现最佳
     缺点：
     - 训练时间较长
     - 需要更多超参数调优
     - 模型复杂度高，推理速度较慢

六、结论与建议
------------------------------------------------------------
  1. CNN-LSTM混合模型在本数据集上表现最佳，准确率达81.29%
  2. 如果注重推理速度和可解释性，推荐使用 RandomForest
  3. 如果需要更好的分类精度，推荐使用 CNN-LSTM
  4. 对于复杂的网络攻击模式，CNN-LSTM 混合模型更具潜力
