# 黄博波 - 个人任务清单 (特征工程)

根据项目分工，黄博波负责 "深度学习" 特征工程相关任务。

## 1. 任务范围

### 1.1 多模态特征融合 (实际: 统计特征 + 时间窗口特征)
- 实现对原始网络流量特征的统计汇总。
- 实现基于滑动时间窗口的动态特征提取。
- 融合多源特征形成增强的特征向量。

### 1.2 特征降维和可视化 (实际: PCA + t-SNE)
- 实现主成分分析 (PCA) 以减少特征冗余。
- 实现 t-SNE 算法对高维流量数据进行 2D/3D 可视化。
- 提供降维前后的性能对比。

### 1.3 特征重要性分析 (实际: RandomForest 特征重要性)
- 利用随机森林算法评估各个特征对攻击识别的贡献度。
- 输出特征重要性排名图表。

### 1.4 建立 "特征数据库" (实际: 特征选择结果保存)
- 持久化保存经过筛选和降维后的最优特征集合。
- 建立特征配置映射表，供检测引擎实时调用。
