## 🚀 部署说明

### 前端部署
1. 确保 `index.html` 和 `script.js` 在同一目录
2. 使用任何Web服务器托管（如 nginx, Apache, 或 Python SimpleHTTPServer）
3. 访问 `http://localhost:端口/index.html`

### 后端要求
1. 后端需要实现 `/api/upload` 接口
2. 接受 `multipart/form-data` 格式
3. 返回 JSON 格式响应

### 示例后端代码（Flask）
```python
from flask import Flask, request, jsonify
import os

app = Flask(__name__)
UPLOAD_FOLDER = 'uploads'

@app.route('/api/upload', methods=['POST'])
def upload_file():
    if 'file' not in request.files:
        return jsonify({'error': '没有文件'}), 400
    
    file = request.files['file']
    
    if file.filename == '':
        return jsonify({'error': '文件名为空'}), 400
    
    if file and allowed_file(file.filename):
        filename = secure_filename(file.filename)
        filepath = os.path.join(UPLOAD_FOLDER, filename)
        file.save(filepath)
        
        return jsonify({
            'success': True,
            'message': '文件上传成功',
            'filename': filename
        }), 200
    
    return jsonify({'error': '文件格式不支持'}), 400

def allowed_file(filename):
    return '.' in filename and \
           filename.rsplit('.', 1)[1].lower() in ['pcap', 'pcapng']
```

## 📞 技术支持

如遇到问题，请检查：

1. **浏览器控制台** - 查看JavaScript错误
2. **网络面板** - 查看API请求状态
3. **后端日志** - 查看服务器错误信息

常见问题：
- **上传按钮不可用** → 检查是否选择了文件
- **上传失败** → 检查后端API是否正常
- **历史记录不显示** → 检查localStorage是否被禁用
- **进度条不动** → 检查网络连接
